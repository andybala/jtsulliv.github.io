It seems as though there is a separation between the people that subscribed, and those that didn't. To more clearly show that, you decide to draw that separating line (also known as the decision boundary).

Now if you were to receive some new data on an article that contains length and number of inbound links, you could predict whether or not a viewer subscribed. For example, a data point above the decision boundary might indicate someone subscribed, while a point below the decision boundary might be someone who did not subscribe.

This is the basic idea behind logistic regression (more specifically binary logistic regression, but we'll get to that in a minute). Given a set of data, you fit a decision boundary such that you can make predictions. Since our predictions are categorical ("yes", "no"), our problem uses a classification model. If we had a different problem where we were trying to predict a numerical outcome (say, housing prices), we could try and use a regression model, such as linear regression.
